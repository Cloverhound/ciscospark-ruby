# Cisco Spark for Ruby

Enjoy a ruby gem for the Cisco Spark API.

## Using an API Call.

1.) add it to your GemFile!

gem 'ciscospark-ruby'

2.) Call it with your access_token

Spark.get_room('token','roomid')

Enjoy!


That's pretty much it!
